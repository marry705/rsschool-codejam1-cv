LoveTrianglesCount Code

The getLoveTrianglesCount function calculates the number of love triangles. The function takes an array of integers as the only parameter. The integer k-th element on n-th place means, that the n-th element loves the k-th element. 
            function getLoveTrianglesCount (preferences = []) {
                let numberTriangles=0;
                let pos=0;
                let value=0;
                let rez=0;
                for (let i = 0; i < preferences.length; i++) {
                    pos=preferences[i]-1; 
                    if (pos < preferences.length && pos>=0) {
                        value=preferences[pos]; 
                        for (let j = 0; j < preferences.length; j++) {          
                            if (preferences[j]===i+1) { 
                                if (value===j+1) numberTriangles++;
                            } 
                        }
                    }
                }
                rez=Math.floor(numberTriangles/3);
                return rez;
            }
